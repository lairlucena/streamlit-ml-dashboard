# Streamlit ML Dashboard

## Overview
An interactive dashboard built with Streamlit to present machine learning results for solar energy production.

## Features
- Interactive data exploration
- Visualization of trends and anomalies
- User-friendly interface for non-technical users

## Key Learning
- Basics of deploying ML applications
- Importance of user interaction
- Transforming static results into dynamic insights

## Run App
streamlit run app.py
